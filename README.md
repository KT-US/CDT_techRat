# 📌 Introduction

**CDT_techRat** is an educational project developed by **Cyber Digital Tree (CDT)**.

The purpose of this project is to help learners understand concepts used in:

* Remote administration
* File management workflows
* Directory traversal
* Client–server communication
* Multi-client handling
* Screen monitoring concepts
* System resource management

This project currently focuses on **Windows operating systems** and future expansion is planned.

# 🖥 Multi Client Handling

The system supports:

* Single client interaction
* Multiple client interaction
* Independent client data management

When operations are performed, data is stored separately for each client.
